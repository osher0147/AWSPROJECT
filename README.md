#myShop

#Instalation
to run local
1. Install xampp
2. Copy project in `c:\xampp\htdocs`
3. Open phpmyadmin in browser by typing `localhost/phpmyadmin`
4. create database named `myshop`
5. import `myshop.sql`  in phpmyadmin
6. run your project at `localhost/myshop`
